AIS -- Third-party notices
===========================

The Windows builds of AIS bundle one third-party component so the program runs
without any separate install. It is NOT part of AIS and is licensed on its own
terms, reproduced/pointed to below.

-------------------------------------------------------------------------------
cygwin1.dll  --  the Cygwin POSIX runtime
-------------------------------------------------------------------------------
The Windows build of AIS is compiled with Cygwin and ships cygwin1.dll beside
ais.exe so it runs on any Windows with no Cygwin installation.

Only the Windows distribution includes cygwin1.dll. The Linux and macOS builds
link the system C library only and bundle no third-party components.
