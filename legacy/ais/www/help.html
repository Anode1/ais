Help

Commands:

default (no command) - the same as help
help - this screen
get - get resources for passed keys
put - add new resources with (under) the passed keys
    By default - 3 Usages are supported:
  • Storing of memos (value is not starting with http://, ftp://, / or \\)
  • Storing of URIs
  • URIs with comments (type URI first and then - comments delimited from URI by space)
delete by keys (&keys="something")
delete by value (&value="someIdOrURI"), where value is uri or auto-generated (if memo) id
dumpindex - dump only index
dumpall - dump all keywords with the assotiated URI's
index - Autoindex all files in directory provided by additional parameter 'dir', recursively
